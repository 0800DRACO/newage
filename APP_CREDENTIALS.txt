📋 FIXING BUILD ERRORS (If Needed)

If deployment shows build errors:

1. Check Build Logs
   App Platform → Deployments → View Logs
   Look for error messages

2. Common Issues & Fixes

   ❌ "Non-Zero Exit Code"
   → Check if artisan command failed
   → May be DB connection timing
   → New startup script handles this
   → Redeploy after code push

   ❌ "Composer install failed"
   → Ensure vendor not in .gitignore
   → Check composer.lock is committed
   → Run composer require locally first

   ❌ "Database connection error"
   → Database loads after application
   → Startup script waits for DB
   → Check DB component is running

3. Redeploy
   $ git push origin main
   App Platform auto-redeploys (5 minutes)
